## How to Use

### For New Users:
1. **Sign Up** - Enter name, email, password
2. **Choose Avatar** - Pick from 12 cartoon options
3. **Start Posting** - Use the new post creator
4. **Edit Profile** - Add location, occupation, website, bio

### For Existing Users:
1. **Login** - Your account works as before
2. **Edit Profile** - Click "Edit Profile" to add an avatar
3. **Choose Avatar** - Select from the avatar grid
4. **Update Info** - Add new fields (optional)
